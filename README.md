# Easy-Converter 🚀

**Easy-Converter** to proste, szybkie i nowoczesne narzędzie internetowe do konwersji plików oraz pobierania audio z platformy YouTube. Strona została zaprojektowana z myślą o maksymalnej prostocie i eleganckim, ciemnym wyglądzie.

[Live Demo](https://ewewerban.github.io/easy-convert/) 🌍

---

## ✨ Funkcje / Features

* **Local File Converter:** Automatycznie wykrywa format dodanego pliku i pozwala na szybką zmianę rozszerzenia (PNG, JPG, MP3, MP4).
* **BETA Downloader:** Umożliwia pobieranie i automatyczną konwersję filmów z YouTube bezpośrednio do formatu MP3 za pomocą jednego kliknięcia.
* **Modern UI:** Czysty, ciemny interfejs z intuicyjnym, rozwijanym menu wyboru formatów.

---

## 🛠️ Jak uruchomić lokalnie / How to run locally

1. Sklonuj to repozytorium lub pobierz plik `index.html`.
2. Otwórz plik `index.html` w dowolnej przeglądarce internetowej (Firefox, Chrome, Edge).
3. Projekt nie wymaga instalowania żadnych dodatkowych bibliotek (czysty HTML/CSS/JavaScript)!
